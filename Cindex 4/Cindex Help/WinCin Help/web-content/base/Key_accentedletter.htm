Entering accented letters and special characters in records

When adding or editing a record you can enter accented characters using the standard keyboard shortcuts used by Microsoft Word.
Special shortcuts as also provided for entering an en-dash and an em-dash.

To enter a letter, first hold how the CONTROL key and type the character that represents the accent (from the table below). Then (with the CONTROL key released) type the letter you want.

Keystrokes Accented letter or character
CONTROL ` (followed by letter) Accent grave over letter
CONTROL ~ (followed by letter) Accent tilde over letter
CONTROL ' (followed by letter) Accent acute over letter
CONTROL ; (followed by letter) Accent umlaut over letter
CONTROL SHIFT 6 (followed by letter) Accent caret (circumflex) over letter
CONTROL , (followed by letter) Accent cedilla under letter
ALT - en-dash
ALT SHIFT - em-dash 

You can also use the Character Map (Character menu) to enter accented letters.
